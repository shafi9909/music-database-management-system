# music-database-management-system
Relational database design with ER modeling, referential integrity and advanced SQL analytics.

## Technologien
- SQL
- Relationale Datenbankmodellierung

## Features
- 1:n und n:m Beziehungen
- Primär- und Fremdschlüssel
- Referenzielle Integrität
- Komplexe Abfragen mit JOIN, GROUP BY, HAVING
- Views zur Datenanalyse

## Beispielabfragen
- Band mit den meisten Songs
- Musiker in mehreren Bands
- Durchschnittliches Erscheinungsjahr pro Genre
